# ▶️ Usage

Run cybersecurity commands and tools from the Linux terminal.

Example:

## 🔍 Nmap Scan

```bash
nmap -sV localhost
```

Used for detecting services and versions running on a target machine.

---

## 🌐 Netcat Listener

```bash
nc -lvp 4444
```

Used to listen on a specific port for network communication testing.
